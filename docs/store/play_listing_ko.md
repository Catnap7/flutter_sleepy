# Google Play Listing (Korean)

## 권장 메타데이터

- 앱 이름: `슬리피: 수면 소리 타이머`
- 짧은 설명: `비, 백색소음, 수면 타이머를 오프라인으로 조용하게 들으세요.`
- 카테고리: `음악/오디오`
- 광고 선언: `광고 없음`
- 데이터 보안: `수집된 데이터 없음`

## 전체 설명

슬리피는 잠들기 전 조용히 틀어두기 좋은 수면 사운드 앱입니다.

원하는 소리를 고르고, 수면 타이머를 설정한 뒤, 화면을 꺼두세요. 슬리피는 회원가입, 스트리밍, 광고 없이 잠들기 전 루틴에 필요한 기능만 간단하게 제공합니다.

### 주요 기능

- 오프라인 수면 사운드 재생
- 비, 핑크 노이즈, 백색소음, 브라운 노이즈, 선풍기 소리, 파도, 캠프파이어, 천둥 소리 제공
- 15분부터 2시간까지 빠른 수면 타이머
- 1분부터 180분까지 직접 설정하는 커스텀 타이머
- 타이머 종료 전 부드럽게 줄어드는 볼륨 페이드아웃
- 화면이 꺼져도 이어지는 백그라운드 재생
- 잠들기 전 긴장을 낮추는 4-7-8 호흡 운동
- 어두운 밤에도 보기 편한 테마와 부드러운 배경 효과

### 이런 상황에 좋아요

- 비 소리나 백색소음을 틀어두고 잠들고 싶을 때
- 주변의 갑작스러운 소리를 부드럽게 가리고 싶을 때
- 공부, 독서, 짧은 휴식에 잔잔한 배경음이 필요할 때
- 일정 시간이 지나면 자동으로 소리가 멈추길 원할 때
- 복잡한 가입이나 설정 없이 바로 사용하고 싶을 때

### 참고

일부 Android 기기에서는 배터리 최적화 설정 때문에 백그라운드 소리가 중단될 수 있습니다. 필요한 경우 앱 안에서 안정적인 백그라운드 재생을 위한 안내를 제공합니다.

슬리피는 휴식과 수면 습관을 돕는 보조 앱이며, 의료기기가 아닙니다. 수면 장애를 진단, 치료하거나 의학적 효과를 보장하지 않습니다.

2. `페이드아웃 수면 타이머` / `15분부터 2시간까지 자동 종료`
3. `화면이 꺼져도 재생` / `잠들기 전 백그라운드 오디오`
4. `잠들기 쉬운 큰 버튼` / `피곤한 밤에도 간단하게`
5. `4-7-8 호흡 운동` / `잠들기 전 천천히 긴장 풀기`
6. `회원가입 없이 바로 사용` / `조용한 밤을 위한 조용한 앱`
