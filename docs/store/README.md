# Sleepy Store Optimization Kit

This folder contains launch-ready templates for improving store listing conversion.

## Files

- `docs/store/play_listing_en.md`: Google Play listing copy and experiment variants.
- `docs/store/play_listing_ko.md`: Korean Google Play listing copy.
- `docs/store/play_console_update_packet.md`: prioritized Play Console update packet.
- `docs/store/play_upload_2026_05/`: upload-ready Google Play phone screenshots.
- `docs/store/scripts/generate_play_upload_assets.py`: regenerates the upload-ready screenshot set.
- `docs/store/app_store_listing_en.md`: App Store metadata and PPO variants.
- `docs/store/benchmark_template.csv`: Competitor benchmark sheet template.

## How to use

1. Apply `play_console_update_packet.md` first so policy, ads, and data-safety claims match the binary.
2. Update localized metadata from `play_listing_en.md` and `play_listing_ko.md`.
3. Replace screenshots using the storyboard in the listing files.
4. Run Play Store listing experiments for title/short-description variants.
5. Track conversion and review sentiment for 14 days.
